# 🎮 Tic Tac Toe AI Game

A modern and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript** with an intelligent AI opponent.

---

## 🚀 Features

* 🤖 Play against **AI (Minimax Algorithm)**
* 👥 Play with **Friend (2 Player Mode)**
* 🎯 **Difficulty Levels**

  * Easy (Random moves)
  * Medium (Smart + Random)
  * Hard (Unbeatable AI)
* ✨ Animated UI with glowing effects
* 🏆 Scoreboard system
* 🔄 Restart game anytime
* 📱 Responsive design (Mobile friendly)

---

## 🧠 AI Logic

The AI uses the **Minimax Algorithm**, which evaluates all possible moves and chooses the best one.

* Hard mode → Always optimal move (almost impossible to beat 💀)
* Medium → Mix of smart + random
* Easy → Random moves

---

## 🕹️ How to Play

1. Open the game
2. Choose mode:

   * Play vs AI 🤖
   * Play with Friend 👥
3. Select difficulty (if AI mode)
4. Click on grid to place your move
5. First to align 3 marks wins 🎉
